# Real-time Chat App without JavaScript/WebSocket

Works on Deno or Node.js with [DenoKV](https://docs.deno.com/kv/manual)

## Running
### on Deno
```bash
$ deno task dev
```

### on Node.js
```bash
$ npm install
$ npm run dev
```

## Compiling Executables
```bash
$ deno task build
$ ./run ## Running app
```
